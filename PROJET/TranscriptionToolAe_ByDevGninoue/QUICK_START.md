# 🚀 GUIDE RAPIDE - Installation en 5 minutes

## Étape 1️⃣ : Installer Whisper (2 min)

### ⏱️ Avant de commencer
- Assurez-vous que **Python 3.8+** est installé
- Vérifier : Ouvrez un terminal et tapez `python --version`

### Installation

#### 🪟 Windows
1. Double-cliquer sur `install_whisper_v2.bat`
2. Attendre que l'installation se termine
3. Appuyer sur une touche quand demandé

#### 🍎 macOS
1. Ouvrir Terminal
2. Naviguer vers le dossier : `cd /chemin/vers/dossier`
3. Exécuter : `bash install_whisper_v2.sh`
4. Attendre la fin

#### 🐧 Linux
Même procédure que macOS (bash install_whisper_v2.sh)

---

## Étape 2️⃣ : Copier le panel After Effects (1 min)

Copier le fichier `TranscriptionTool_v2.jsx` vers :

### 🪟 Windows
```
C:\Program Files\Adobe\Adobe After Effects 2024\Support Files\Scripts\ScriptUI Panels\
```

(Remplacer "2024" par votre version d'After Effects)

### 🍎 macOS
```
/Applications/Adobe After Effects 2024/Scripts/ScriptUI Panels/
```

---

## Étape 3️⃣ : Redémarrer After Effects (1 min)

1. Fermer After Effects complètement
2. Rouvrir After Effects
3. Aller dans : **Fenêtre > Transcription Tool v2.0**

✅ **C'est prêt !**

---

## 🎯 Première utilisation

1. **Ouvrir une vidéo** dans After Effects
2. **Créer une composition** avec la vidéo
3. Cliquer sur **"📤 Exporter + Transcrire"**
4. Choisir où sauvegarder l'audio WAV
5. **Attendre** que Whisper transcrive (peut prendre 1-5 min selon la durée)
6. **Les sous-titres apparaissent automatiquement !**

---

## ❌ Problème ?

### Python non trouvé
```bash
# Télécharger depuis : https://www.python.org/downloads/
# Cocher "Add Python to PATH" à l'installation
```

### Whisper non trouvé après installation
```bash
# Redémarrer votre terminal/PowerShell
# Essayez à nouveau : pip install openai-whisper
```

### Panel n'apparaît pas dans After Effects
- Vérifier le chemin de copie (voir Étape 2)
- Redémarrer After Effects
- Vérifier que le fichier s'appelle bien `TranscriptionTool_v2.jsx`
