# Transcription Tool pour After Effects

## Installation
1. Copiez `TranscriptionTool.jsx` dans le dossier `ScriptUI Panels` d’After Effects :
   - Windows : `C:\Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\ScriptUI Panels\`
   - macOS : `/Applications/Adobe After Effects <version>/Scripts/ScriptUI Panels/`
2. Lancez `install_whisper.bat` (Windows) ou `install_whisper.sh` (Mac/Linux) pour installer Whisper.
3. Redémarrez After Effects.

## Utilisation
- Ouvrez le panneau via **Fenêtre > Transcription Tool**.
- Cliquez sur **Exporter + Transcrire** pour générer un fichier WAV, lancer Whisper et importer automatiquement les sous‑titres.
- Vous pouvez aussi importer un fichier `.srt` manuellement avec le bouton **Importer SRT**.

## Prérequis
- After Effects CC 2020 ou plus récent.
- Python installé sur votre machine.
- Whisper installé via pip.
