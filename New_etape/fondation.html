Html est un language de balisage utulise pour structure le contenue d'une page web.
les balise fondamentale sont :

  • les balise titre de h1 a h6
  • la balise a href
  • la balise p pour les paragraphes
  • la balise image pour les images
  • la balise ul pour les listes non ordonnée et ol pour les listes ordonnées
