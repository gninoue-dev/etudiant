Fomulaire

Un formulaire sert a l'interactions des utuliateurs en entrants des données exemple nom , preom, age.

la balise pour faire un formulaire est < forme > et ces attributs courants sont :
name qui identifie le nom unique du formulaire dans la même page
methode indique pas quel moyens les données serons envoyé ="get" ou ="post".
action Spécifie l’URL vers laquelle les données du formulaire seront envoyées lors de la soumission. Exemple : action="traitement.php"
