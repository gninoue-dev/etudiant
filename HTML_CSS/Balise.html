Balise Paire


Une balise paire est entourée de deux chevron
paire un chevron ouvrant et un chrevons fermant. < > est une balise ouvrant et < /> est une balise fermant. balises HTML paires (c’est-à-dire avec une balise ouvrante et une balise fermante) :
< html > : C’est la racine d’un document HTML. Elle englobe tout le contenu de la page.
< head > : Contient les métadonnées, les liens vers les fichiers CSS, les scripts, le titre, etc.
< title > : Définit le titre de la page affiché dans l’onglet du navigateur.
< bod Voici une liste des principales y > : Contient tout le contenu visible de la page web.
< h1 > à < h6 > : Représentent les titres, du plus important < h1 > au moins important < h6 >.
< p > : Définit un paragraphe de texte.
< a > : Crée un lien hypertexte vers une autre page ou ressource.
< div > : Sert à structurer la page en blocs. Très utilisé pour la mise en page.
< span > : Conteneur en ligne utilisé pour styliser ou cibler une portion de texte.
< ul > : Liste non ordonnée (avec des puces).
< ol > : Liste ordonnée (avec des numéros).
< li > : Élément d’une liste utilisé dans `< ul > ou < ol >.
< table > : Crée un tableau.
< tr > : Définit une ligne dans un tableau.
< td > : Définit une cellule dans une ligne de tableau.
< th > : Définit une cellule d’en-tête dans un tableau.
< form > : Crée un formulaire pour la saisie de données.
< label > : Étiquette associée à un champ de formulaire.
< textarea > : Zone de texte multiligne.
< select > : Menu déroulant.
< option > : Option dans un menu déroulant.
< section > : Regroupe un contenu thématique.
< article > : Contenu autonome (comme un article de blog).
< header > : En-tête d’une section ou de la page.
< footer > : Pied de page ou de section.
< nav > : Zone de navigation (menu).
< main > : Contenu principal de la page.
< figure > : Encadre une image ou un média avec sa légende.
< figcaption > : Légende associée à une image ou un média.
< video > : Intègre une vidéo.
< audio > : Intègre un fichier audio.
< script > : Intègre du code JavaScript.
< style > : Intègre du CSS directement dans la page.


Balise Orpheline

les balise orpheline < > servent le plus souvent à inseret des truc a un endroit précis le / en fin n'est pas obligatoire comme avec les balises paires.
voici la liste de balise orpheline :
< br > : Insère un saut de ligne dans le texte.
< hr > : Insère une ligne horizontale pour séparer visuellement des sections.
< img > : Affiche une image dans la page. Nécessite l’attribut src pour indiquer la source.
< input > : Crée un champ de saisie dans un formulaire (texte, bouton, case à cocher, etc.).
< meta > : Fournit des métadonnées sur le document HTML (encodage, auteur, description, etc.).
< link > : Sert à lier des ressources externes comme des feuilles de style CSS.
< source > : Définit une source pour les éléments < video > ou < audio >.
< area > : Définit une zone cliquable dans une image utilisée avec ``.
< base > : Spécifie l’URL de base pour toutes les URL relatives dans le document.
< col > : Spécifie les propriétés d’une colonne dans un tableau, utilisé avec ``.
< embed > : Intègre du contenu externe comme des vidéos ou des animations Flash.
< track > : Ajoute des sous-titres ou des descriptions à une vidéo.
< wbr > : Indique un point potentiel de retour à la ligne dans un mot long.
< param > : Définit un paramètre pour un objet multimédia inséré avec `
